# PhonePe Transaction Analysis Dashboard (Power BI)

1.Project Overview
This project presents an interactive Power BI dashboard analyzing PhonePe transaction data. The goal is to uncover meaningful insights about transaction patterns, user behavior, and regional performance across India.

---

2. Objective
To analyze digital payment trends and identify:
- Transaction growth patterns
- State-wise performance
- User engagement behavior
- Key factors influencing transaction volume

---

3. Dataset Description
The dataset includes:
- Transaction data (amount & count)
- State-wise and district-wise information
- User activity metrics
- Time-based trends

---

4. Tools & Technologies
- Power BI (Data Visualization)
- Excel / CSV (Data Source)
- Data Cleaning & Transformation (Power Query)

---

5. Dashboard Features
- Transaction trends over time
- State-wise transaction analysis (map visuals)
- Comparison of transaction count vs amount
- User engagement insights
- Drill-down capability for deeper analysis

---

6. Key Insights
- Certain states dominate in both transaction count and value
- High transaction volume does not always mean high transaction value
- User engagement varies significantly across regions
- Digital payment adoption is increasing over time

---

7. Business Insights
- Helps identify high-performing regions for targeted marketing
- Useful for improving user engagement strategies
- Assists in decision-making for expanding digital payment services

---

8. How to Use
- Download the `.pbix` file from this repository  
- Open using Power BI Desktop  
- Interact with filters and visuals
